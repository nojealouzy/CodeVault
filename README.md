# ⚡ CodeVault — DSA Journey Tracker

> A stunning, action anime-themed web app to track your DSA problem-solving journey across LeetCode, HackerRank, CodeForces, and more.

![CodeVault Preview](https://img.shields.io/badge/Theme-Action%20Anime-blue?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=for-the-badge)

## 🔥 Features

- **Dashboard** — Stats cards, activity heatmap, streak counter, recent problems, platform breakdown
- **Problem Vault** — Filterable grid/list view with search across all your solved problems
- **Add Problem** — Log problem title, platform, difficulty, topic, question, approach, solution code, complexities & notes
- **Analytics** — Donut chart, topic mastery bars, monthly progress, performance stats
- **Global Search** — Press `Ctrl+K` to instantly search across all problems

## 🎨 Action Anime Theme

- ⚡ Rising fire embers (electric blue, fire orange, gold)
- ⚡ Horizontal energy sparks shooting across the screen
- ⚡ Random lightning bolt flashes
- ⚡ Electric glow on hover interactions
- ⚡ Dark intense color palette with neon accents

## 🚀 Quick Start

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/CodeVault.git
   cd CodeVault
   ```

2. Open `index.html` in your browser — that's it! No build tools needed.

   Or serve locally:
   ```bash
   npx serve .
   ```

## 🔒 Privacy

All data is stored in your browser's **localStorage**. Nothing is sent to any server. Your problems and solutions are 100% private and stay on YOUR device.

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, animations, glassmorphism
- **Vanilla JavaScript** — Canvas animations, localStorage CRUD, dynamic charts
